## 👨‍💻 About Me

I am an AWS Cloud and DevOps engineer with **8+ years of experience** across cloud infrastructure, automation, CI/CD, security, migrations and operational support. My core strength is turning repetitive or risky infrastructure work into standardized, reviewable and reliable delivery workflows.

I work primarily with **AWS, Terraform, CloudFormation, GitHub Actions, Python, Bash, Docker, ECS and Fargate**. My Kubernetes/EKS experience is focused on Terraform-based cluster provisioning and infrastructure management.
